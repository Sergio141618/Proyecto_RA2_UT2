# 05 — Instalación de Odoo

1. Ejecuta el instalador (`.exe`) como **Administrador**.

   ![Ejecutar instalacion](../assets/img/05-instalacion_odoo/Ejecutar_instalador.png)
2. Acepta la licencia y elige **ruta de instalación**.

   ![Aceptar licencia](../assets/img/05-instalacion_odoo/Aceptar_licencia.png)

   ![Ejecutar instalacion](../assets/img/05-instalacion_odoo/Instalar.png)
3. Si se solicita, configura la conexión a **PostgreSQL**.

   ![Configuracion de conexion a PostgreSQL](../assets/img/05-instalacion_odoo/Conexión_PostgreSQL.png)
4. Completa la instalación y finaliza el asistente.

> Resultado esperado: Odoo instalado, acceso a accesos directos/servicio.
